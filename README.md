# Meal Analyzer App

This is a mobile/app application that helps users analyze their meals and track nutritional information.

## Features
- Analyze food items using AI-powered functions
- Display nutrition results
- Track meal history
- Built with TypeScript, React, and Supabase

## How to Run
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

## Technologies
- React (TypeScript)
- Supabase for backend and database
- Tailwind CSS for styling
